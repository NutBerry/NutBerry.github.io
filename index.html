NutBerry

Offroading Ethereum Transactions

State minimized EVM,

Taking contracts smart off the road.

The NutBerry project is here to scale

Ethereum transactions by a permissionless

smart contract which is able to replay and verify

transactions on-chain and therefore guarantee consensus.

NutBerry makes it possible to improve the

thoughput of trustless interactions and greatly

scales the transaction capacity for

applications with a trusted multi-party architecture.

NutBerry is able to keep track of ERC20 and ERC721 tokens.

With future support of additional token standards like

ERC1948 and ERC1949.
